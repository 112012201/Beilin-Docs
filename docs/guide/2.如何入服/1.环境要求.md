> [!CAUTION]
> 部署环境时未保存的文件可能损坏。

> [!WARNING]
> 以下示例所需系统为Windows10 1909+,Windows11 22H2+

Minecraft的1.19.xJAVA版本要求是 ==JAVA17+== \
本档以Zulu17为例
## 下载
首先你需要下载Zulu17 [前往下载Zulu17](https://www.azul.com/core-post-download/?endpoint=zulu&uuid=c0bbc086-86ce-4aa9-a815-040c507644f2) \
下载完文件应该会在Downloads文件夹内如果没有那就是在你自己设置的下载文件夹

::: file-tree icon="colored"
- Downloads
  - zulu17.64.17-ca-jdk17.0.18-win_x64.msi
- …
:::
## 安装
双击运行安装文件一般直接Next就行如需要改安装路径在下图位置时点击Browse…更改
![QQ20260218-173529-05dc3b3cd4ba1d7719cdf9589d6b5413.png](https://t.tutu.to/img/u0DBK) \
出现下图页面时点击Install安装就会开始 \
![Install-4d7d4f0ceb1e73743f43a372e6598156.png](https://t.tutu.to/img/u0Knj) \
安装完成直接点Finish安装程序就会退出
## 结束
至此环境已经安装完成了，请往后阅读查看后续步骤
