## Ledger i 使用
Ledger i用于查找被玩家手动破坏的方块
如果用此参数发现了破坏者请截图 附带坐标 被破坏情况 然后其余等管理组处理然后管理组会为你rollback
## Ledger s 使用
> [!CAUTION]
> 恶意使用 s 参数卡服者将被严肃处理!

如果你使用了i参数没发有修改但确实被破坏或修改了可使用此参数 \
此参数分为多个后缀 \
action:<修改的什么状态> \
after:<时间(d,h,m,s)> \
before:<你不需要知道> \
object:<方块ID> \
range:<大小> \
source:<玩家ID> \
world:<你不需要知道> \
本服仅允许玩家使用action:after:object:range:参数 \
我们推荐查找使用 /lg s object:<方块ID> after:<时间> action:<修改的什么状态> \
然后系统会弹出被修改的方块点击旁边的坐标看看是不是你被破坏的 \
当你看见一个有这@开头的玩家修改了你的方块那么那是WorldEdit修改的无法继续追查 直接找管理组RollBack即可 \
如果你发现是正常的玩家ID那么使用/lg s source:<玩家ID> after:<时间> action:<修改的什么状态> 确认此玩家修改的方块 \
然后把修改信息截图发送之管理组后你就可以继续建设了,管理组会处理完后续事情并帮你rollback
